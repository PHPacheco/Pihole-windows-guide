# Project Overview
This project provides a step-by-step tutorial for deploying a Pi-hole instance to act as a network-wide ad blocker.

- Implementation Time: 1 to 3 hours.

- Post-Installation (Filters & Clients): 15 to 30 minutes.

## Prerequisites
To ensure a stable installation, the following hardware is recommended:

- **Raspberry Pi**: Raspberry Pi Zero 2 W (recommended for its balance of power and efficiency).

- **MicroSD Card**: Minimum 8GB (~3.5GB for OS/Pi-hole and extra packages, the remainder for logs and system longevity).

- **Power Supply**: Micro-USB to USB cable (5V) and a 5V power adapter.

Computer: Required for initial flashing and remote configuration.

Note: 8 GB is sufficient, 16 GB or 32 GB (Class 10 / A1) cards are recommended.
